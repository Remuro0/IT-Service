# 🖥️ ИТ-Сервис — Система управления инфраструктурой

Профессиональная платформа для управления ИТ-инфраструктурой, безопасностью и поддержкой 24/7.

## 📋 Возможности

- 🔐 Многоуровневая авторизация (User, Engineer, Manager, DB_Admin, Admin)
- 🗄️ Управление базами данных (Local + Global синхронизация)
- 📊 Мониторинг системной нагрузки в реальном времени
- 🛒 Корзина и биллинг услуг
- 📋 Система логов с настраиваемым хранением
- 🔔 Уведомления пользователей
- 🖼️ Редактирование профиля с загрузкой аватарок
- 🧩 Капча-пазл для защиты от ботов
- ⏱️ Таймауты сессий по ролям

## 🚀 Быстрый старт

### Требования

- PHP 7.4+
- MySQL 5.7+ / MariaDB 10.3+
- XAMPP / OpenServer / Apache + Nginx
